Chapter 2 Exercises
-------------------
- Which of the following will match?
```
  [a,b,a]=[1,2,3]
  [a,b,a]=[1,1,2]
  [a,b,a]=[1,2,1]

  Ans: [a,b,a] =[1,2,1]
```

If you assume the variable a initially contains the value 2 ,which of the
following will match?
```
  [a,b,a]=[1,2,3]
  [a,b,a]=[1,1,2]
  a=1
  ^a=2
  ^a=1
  ^a=2-a

  Ans: ^a=2
```
